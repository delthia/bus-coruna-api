Historia

Mapa con el que se inicia el proyecto

A partir de ahí, la parte más complicada estaba hecha, ahora ya sabía cómo descargar los datos y cómo manipularlos para relacionar una parada con las líneas que pasan por la misma, así que solo quedaba aplicar lo que conocía de Flask para hacer que se pudiera acceder a toda la información de los buses desde una página web.

De esta manera, pude empezar a poner en práctica lo que había aprendido de flask siguiendo este tutorial de python. Historia

This proyect came up in September 2022, when I had the idea to make a map with all the stops for the bus of Coruña, as an example of using leafletjs, and to learn to create GeoJSON in python.

Mapa that starts the project

From there, the most difficult part was done, and now I new how to download the data and manipulate it to connect an stop to the lines that go through it, so I just had to apply what I knew about Flask to make it possible to get the information through a webpage.

This way, I was able to start put into practice what I had learned about flask following this python tutorial. I thought that this was an interesting project because I had to learn to use an API in a webapp that generated static pages with the data, but that also allowed to refresh the data dynamically. To do this I needed to learn a bit about JavaScript, something I had never done. I also had to research how to make a webapp installable as a pwa, following this guide.

This page is far from done, there is still some features to be implemented, as the departures table, the realtime position of the buses, as well as make the PWA better by allowing to set favourite stops, select a language that is remembered between uses, set an alert for an specific bus or make the stop search better, but it is already in an state that is usable.

This is my first project that includes my knowledge in several areas

  • Python programming
    • Working with JSON
    • Generating an API
  • Web design: HTML, CSS and JS
  • Docker, where the program is executed
