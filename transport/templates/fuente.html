Fuente del proyecto

Este es un proyecto libre que muestra información en tiempo real sobre el bus de Coruña. Los datos se descargan desde su servidor y se generan algunos archivos que continenen las paradas, las líneas, una relación de estas y un archivo con los datos para el mapa, que se actualizan periódicamente. El servidor también renderiza las páginas con la información dinámica, como los buses para una parada, y hospeda una API que los clientes utilizan para conseguir los últimos datos.

Todo está hecho en python, utilizando la librería Flask, una librería para el lenguaje de programación Python. El backend es sencillo, ya que solo tiene las funciones que se mencionan arriba. Lo más complicado son las plantillas utilizadas para renderizar la página, ya que está disponible en tres idiomas y las plantillas deben de funcionar para todos ellos, así como los scripts del frontend, ya que utilizar javascript no es algo a lo que esté acostumbrado.

Después la aplicación se mete en un contenedor de Docker utilizando gunicorn, ya que esto me permite instalar y actualizar fácilmente la aplicación en el servidor simplemente ejecutando un script que se incluye en el repositorio.

El código estará publicado en GitHub bajo la licencia AGPL-v3, que está hecha específicamente para software de servidor, de modo que cualquiera puede modificar y ejecutar el programa, siempre y cuando los usuarios sigan teniendo acceso al código fuente de la versión modificada que está utilizando.

Source for the project

This is a free project that shows realtime information on the bus of Coruña. The data is gathered from their server, and some files that contain the stops, the lines, a relation of both and a file with the data for the map is created and update periodically. The server also renders the pages with the dynamic information such as the buses for an stop, and hosts an API that the client uses the fetch the latest data.

It is all made in python, using Flask, a library for the Python programming language. The backend part is simple, as it just does the functions mentioned above. The more complex part are the templates used to render de page, as this page is available in three languages, and the templates should work for all of them, as well as the scripts for the frontend, as using javascript is no something I'm used to doing.

The application is then put into a Docker container, using gunicorn, as this allows me to easily install and update it in the server with just executing an script included on the repo.

The code will be published to GitHub under the AGPL-v3 license, which is specifically made for server software, so that anyone can modify and run the program, as long as the users still have access to the source for the modified version they are using.
