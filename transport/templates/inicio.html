bus.delthia.com - Información en tiempo real sobre el bus de Coruña

Información de parada

Consulta información sobre los buses que están a punto de llegar. Esta página también incluye una tabla que muestra las características de la parada, con datos a partir de OpenStreetMap, y un mapa que muestra dónde se encuentra la parada.

Mapa de paradas

Consulta un mapa con todas las paradas, toca en una parada para ver las líneas que pasan por ella o simplemente comprueba qué paradas hay en una zona.

Ver el mapa

Información de línea

Diagrama con todas las paradas de una línea, la posición del bus en el recorrido (próximamente) y los horarios de salida de la línea. También muestra los enlaces con otras líneas en cada parada.

Ver la lista de líneas

Software libre

El código de esta página está disponible bajo una licencia libre, para que puedas ver como funciona y modificarlo.

La página está hecha con python, utilizando la librería flask. Descarga los datos desde la página de iTranvías, los adapta para trabajar con ellos y genera las páginas con la información de los buses y de otras fuentes. También incluye una api que se utiliza para actualizar los datos sin tener que recargar la página.
