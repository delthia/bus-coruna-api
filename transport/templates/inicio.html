Bus Urbano de Coruña

Esta página incluye la información en tiempo real sobre el bus urbando de coruña de iTranvías, pero también incluye datos de las paradas a partir de OpenStreetMap. Este es mi primer proyecto "funcional", y consiste en esta página hecha con flask, una librería para python. Si tienes curiosidad, te gustaría aprender, o crees que puedes ayudarme a solucionar errores o mejorar la página, puedes ver, modificar y aportar al código que se aloja en Codeberg y GitHub.
